INSTRUCCIONES PARA SUBIR A GITHUB PAGES (LINK EDITABLE)
1) Crea una cuenta en GitHub y un repositorio público (ej: igss-mazatenango-bolsa).
2) Sube todos los archivos de esta carpeta al repositorio (index.html).
3) Ve a Settings -> Pages, selecciona Branch: main y Root folder -> Save.
4) GitHub generará un enlace: https://TUUSUARIO.github.io/igss-mazatenango-bolsa
5) Para editar, modifica index.html desde GitHub (Edit file) o sube cambios desde tu máquina.

Si prefieres, puedes usar Netlify Drop o Vercel para desplegar en segundos arrastrando este folder.
